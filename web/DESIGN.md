# VelopX Web Design Tokens

Hybrid design system: **shadcn/ui** (Radix primitives) + **ui-ux-pro-max** guidance.

## Palette

| Token | Hex | Usage |
|-------|-----|-------|
| navy-950 | `#070C14` | App background |
| navy-900 | `#0C1526` | Cards, sidebar |
| navy-800 | `#111E34` | Elevated surfaces |
| orange-500 | `#F5A623` | Primary accent, CTAs |
| foreground | `#E8ECF1` | Primary text |
| muted-foreground | `#8A97AA` | Secondary text |

## Typography

- **Headings:** Space Grotesk (`font-display`)
- **Body:** Inter (`font-sans`)

## UX rules (ui-ux-pro-max)

- Minimum touch target 44px on mobile nav
- Status always shown as colored badge, never plain text
- Empty and error states required on every data fetch
- Dark mode only — no light theme toggle in Phase 1
