GET /v1/parts/benchmark Stable

Returns benchmark pricing — low, average, and high — for a part across all conditions. Used by insurance assessors for invoice validation and by dealers or garages to verify competitor quotes.

Request
// GET /v1/parts/benchmark // ?oem=53711-42200&country=KE // Authorization: Bearer {api_key}
Query Params
oem string OEM part number (required) country string ISO 3166-1 alpha-2 (KE, ZA...) condition string oem|aftermarket|used|all
